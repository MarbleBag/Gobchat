1. Use atom to turn relevant files (.md) in /docs into pdfs
	1. .md -> markdown preview
	2. Export as html
	4. Open in chrome
	5. Print as pdf to /docs
2. Run buildscript.bat
3. The new release will be moved to the root folder (gobchat-{version}.zip) alongside a debug release (debugging symbols)
4. Put up a new release on github.com
	1. Release title is v{version}
	2. Description is used for patch notes
	3. Updater will download the gobchat-{version}.zip attachment
	4. Updater will mark pre-releases as 'beta'
